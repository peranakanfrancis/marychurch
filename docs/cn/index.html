教友祈祷文

慈爱的天父、感谢祢！清晨、唤醒我们，让我们在祈祷中与祢相遇！黑夜已经过去，新的一天，求祢快到我们心里来，驱散我们心中一切的苦闷、和不愉快的情绪，让黑暗远离我们，让喜乐和平安，充满我们的心！ 主，祢爱所有人，祢渴望、我们每个人都能平平安安、灵魂得救！

祢告诉我们说：有一百只羊，如果其中一只迷了路，祢也会放下那九十九只，去寻找那一只迷失的羊（18:12-14）。 是的主，我们都是迷失的羔羊。世界再美好，将来有一天，我们都要离开！救灵魂，才是今生最重要的事。但很多时候，我们在生活、家庭、工作、救灵魂这些事情中，总是把救灵魂的事，排在最后。追求生活的幸福，祢并不反对。但是，把世界当成永久的家，把心思都用在讨好肉身上，忘了救灵魂，就是我们最大的糊涂。

现在、是圣母月。恩典的时期，求祢赐给我们生活中的恩典，更赐给我们灵魂得救的恩典。今天，让我们把心敞开，完全属于祢。也让我们的家庭，无论大人还是孩子，都能全心归向祢。求祢把我们这些迷失的羔羊都抱在祢的怀里，将来灵魂得救，在天上、永远赞美祢。祢是天主，永生永王，阿们。
